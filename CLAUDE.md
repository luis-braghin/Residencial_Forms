# CLAUDE.md — Residencial Forms

## Regras obrigatórias
- Sempre que alterar funcionalidades ou dependências, atualize o README.md
- Nunca commite arquivos .env ou chaves de API
- Mantenha consistência com o design system documentado em IDENTITY.md (Port Go branding)
- Ao adicionar uma variável de ambiente nova, adicione também no README.md na seção "Variáveis de ambiente"

## Stack
- HTML/CSS/JavaScript puro (sem frameworks)
- Formulário estático
- Design system Port Go (Plus Jakarta Sans, HSL colors, dark theme, glassmorphism)
- Deploy via hosting estático

## Contexto
Formulário de cadastro de telefone para o Residencial Imperatriz. Segue o design system do Port Go com visual escuro e glassmorphism. Utilizado para registro de moradores no sistema de comunicação do condomínio.
